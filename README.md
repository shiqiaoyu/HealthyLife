# HealthyLife 健康生活助手
# 作者：石乔宇
# 学号：42311183

## 项目简介
一款专注于日常健康管理的 Android 应用，提供健康打卡、心情记录、BMI计算与健康知识推送功能。

## 功能模块
1. 健康打卡
2. 心情记录
3. BMI 计算器
4. 健康知识推荐
